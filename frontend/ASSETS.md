# Assets Setup

For development, assets are optional. For production builds, you'll need to add:

## Required Assets for Production

1. **icon.png** - App icon (1024x1024px)
2. **splash.png** - Splash screen (1242x2436px for iOS, 1080x1920px for Android)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024px)
4. **favicon.png** - Web favicon (48x48px)
5. **notification-icon.png** - Notification icon (96x96px)

## Quick Setup

You can use placeholder images for now, or generate them later using:
- Expo's asset generator
- Online icon generators
- Design tools

For now, the app will work without these assets in development mode.
